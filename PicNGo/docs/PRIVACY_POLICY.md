# PicNGo Privacy Policy

**Effective date:** March 4, 2026

PicNGo (“we”, “our”, “us”) helps users analyze food photos with AI. This Privacy Policy explains what data PicNGo processes, how it is used, and when it is shared.

## 1. Data We Process

When you use analysis features, PicNGo may process:

- Food photos you capture or select
- Text sent for analysis (for example, food/ingredient prompts)
- App language preference used for AI response formatting
- Your OpenAI API key (entered by you)

## 2. How Data Is Collected

- Photos are provided directly by you through camera or photo library selection.
- Text prompts are generated by the app based on your requested analysis.
- Language preference is stored in app settings.
- API key is entered by you in settings.

## 3. How Data Is Used

PicNGo uses the above data to:

- Send requests for food and ingredient analysis
- Return nutrition and health-related insights in the app
- Format responses in your selected language

## 4. Third-Party Sharing

PicNGo shares analysis request data with:

- **OpenAI** (AI service provider) to generate food and ingredient analysis results

Data sent to OpenAI can include selected photos and text prompts related to analysis. Requests are only sent after user permission is granted in the app.

## 5. User Permission and Control

- PicNGo asks for explicit in-app permission before sending analysis data to OpenAI.
- You can revoke permission anytime in **Settings > Data & Privacy**.
- If permission is not granted, PicNGo does not send analysis data to OpenAI.

## 6. Local Storage

PicNGo stores certain settings locally on your device, including:

- OpenAI API key
- Language preference
- AI data-sharing permission preference

## 7. Data Retention

PicNGo does not operate its own backend for storing your analysis data. Data handling by OpenAI is subject to OpenAI’s policies and terms.

## 8. Children’s Privacy

PicNGo is not directed to children under 13. If you believe a child provided personal data, contact us and we will help remove locally stored app data guidance.

## 9. Changes to This Policy

We may update this policy from time to time. Updated versions will include a new effective date.
