# App Review Response Template (Guidelines 5.1.1(i) and 5.1.2(i))

Hello App Review Team,

Thank you for your feedback.

We have updated PicNGo (version 1.0 resubmission) to address Guidelines 5.1.1(i) and 5.1.2(i):

1. **Clear in-app disclosure before data sharing**
- Before any analysis request is sent, the app now shows an explicit permission prompt.
- The prompt explains what data is sent (selected photo, food/ingredient text prompts, language preference), who receives it (**OpenAI**), and why (food analysis features).

2. **User permission before transmission**
- No analysis request is sent to OpenAI unless the user taps consent.
- If consent is not granted, data is not transmitted.

3. **User control in Settings**
- Added **Settings > Data & Privacy** toggle to allow/revoke AI data sharing at any time.

4. **Privacy Policy update**
- The privacy policy now clearly describes:
  - what data is collected/processed,
  - how data is collected,
  - how data is used,
  - and that data is shared with OpenAI for AI analysis.

In App Store Connect, we also updated:
- Privacy Policy URL
- App Review Information notes to explain the in-app consent flow and third-party AI usage.

Please let us know if any additional clarification is needed.

Best regards,
PicNGo Developer
